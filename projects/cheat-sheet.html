Ruby Basics

Methods

Methods are created using "def" followed by the method name. Method names should begin with a lowercase letter. Methods can take any number of arguments and the arguments can be given default values as well.

Classes

Classes are created using "class" followed by the class name. Class names should begin with a uppercase letter. Classes can contain a multitude of methods and have instance variables which can be used throughout all the methods within the class. There are also reader and writer attributes that can be applied to classes that allow interaction with the variables outside of the class.

Array

Arrays are a collection of objects ordered by an index. The first object is indexed at 0 and the last can be indexed at -1.

Hashes

Hashes are a collection of unique keys and their values. You can access values by calling upon the keys.

Strings

Strings are an object which can be made up of most things held within single or double quotes.

Integers

Objects that are whole numbers.

Boolean expressions

a == b

Returns true when a is equal to b.

a != b

Returns true when a is not equal to b.

a > b

Returns true when a is greater than b.

a < b

Returns true when a is less than b.

a >= b

Returns true when a is greater than or equal to b.

a <= b

Returns true when a is less than or equal to b.

Ruby Operators

+

For adding the values on both side of the operator

-

For subtracting the values on both side of the operator

*

For multiplying the values on both side of the operator

/

For dividing the values on both side of the operator

%

For finding the remainder of the values on both sides of the operator

&&

If both sides of the operator return true, then the condition is true.

||

If either side of the operator return true, then the condition is true.

!

Reverses the logical state. If a condition is true, it will be false and vice versa.

Array methods

collect

Passes given elements to the block. Creates new array containing new values. Syntax and examples

concat

Adds elements of given arrays together. Syntax and examples

count

Returns the number of elements unless an argument is given. If an argument is given, returns the number of elements equal to the given argument. Syntax and examples

delete_if

Deletes every element where the given block is true for. Syntax and examples

each

Performs the given block over each element. Syntax and examples

each_index

Performs the given block over each index. Syntax and examples

flatten

Returns a new array that is a one dimensional array of all the elements in a given array. Syntax and examples

include?

Returns true if the given object is present. Syntax and examples

insert

join

Returns a string created by converting all the elements of the array to a string and joining them together. Syntax and examples

length

Returns the number of elements in a given array. May be equal to 0.Syntax and examples

map

Returns the result a given block into a new array. Similar to collect.Syntax and examples

reverse

Returns a new array containing the elements in reverse order.Syntax and examples

sample

select

Returns the elements for which the given block are true.Syntax and examples
