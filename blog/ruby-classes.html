Ruby Classes

What are They?

1/17/2016

Today we're going to talk a little about classes. Classes are objects in Ruby that can contain multiple methods all under one name. I created a simple calculator class here.

In the beginning method, we take two variables x and y and make them equal to @var1 and @var2. These are isntance variables that allow them to be used anywhere in the class. This class named calculator can be called upon with the name of the variable you want using the variable = to the class.new(with any variable required here). An example would be numbers = Calculator.new(4,2). Now you can call any of the methods under the calculator class for the numbers variable and it will return you an answer relative to the variable it has received. If you were to call the add method on numbers, you would use numbers.add and it would return 6. All of the methods can be used that way.

Another special thing about class are isntance methods. In the doublesum method we call upon the add method and double the value the add method returns. Using classes allows us to use what other methods return in other places as a way to simplify things. Classes are a very useful organization tool allowing us to group together methods that have similar purposes under one "roof". For more information on Ruby Classes, http://www.tutorialspoint.com/ruby/ruby_classes.htm has some more.
