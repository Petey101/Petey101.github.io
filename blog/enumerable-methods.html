Map Method

1/10/2016

Today we're going to talk about a specific method that is extremely useful for many of the functions that you will possibly want to do in Ruby. The map method is used to output the results of a block or code you give it and displays it in a new array.

In the example above, I defined an array with the integers 1,2,3, and 4 inside it. I then proceeded to use the array method to pass each element of "array" into the block {|x| x+1}. This block will take each element of "array" as x and for each x it will do the function "x+1". I made the output of array.map {|x| x+1} equal to new_array and if I were to call upton new_array, I would get an array with the elements 2,3,4,and 5. This is an extremely simple use for the map method, but I hope it opens you up the the countless possibilities it can be used in!
